Atlanta Workspace Directory

This is a list of workspaces I found when I was looking for office space. Send pull requests to add any workspaces I missed.

Criteria for Inclusion

  • This is not craigslist, only permanent shared and coworking spaces are included.
  • Locations should have a significant focus on technology companies.
